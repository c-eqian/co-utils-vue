Function useSort

  • useSort<T>(data, options?): T[]

  • 数据排序

    Type Parameters

    • T = any

    Parameters

    • data: T[]
    • Optional options: ISort<T>

    Returns T[]

    Example

    // 排序数组
    useSort([4, 8, 3, 4, 5, 7, 9, 40, 520], {
            order: 'dec'
          }) // [520, 40, 9, 8, 7, 5, 4, 4, 3]
    useSort([4, 8, 3, 4, 5, 7, 9, 40, 520], {
            order: 'asc'
          }) // [3, 4, 4, 5, 7, 8, 9, 40, 520]
    // 排序对象数组
          useSort(
            [
              {
                name: '2',
                age: 19
              },

                age: 18
              }
            ],
            {
              order: 'dec',
              key: 'age'
            }
          ) // [
