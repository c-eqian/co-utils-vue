co-utils-vue

介绍

co-utils-vue工具库是基于typescript、vue3实现的一个常用库封装

安装

npm install co-utils-vue -D

使用

此使用示例可能与实际未同步、存在差异，建议请阅读实时使用文档

Hooks

usePick

与Pick选类似，但是usePick取指定一组属性值，返回一个新的属性值

const test = {
    book: '倚天屠龙记',
    name: '金庸',
    price: '18',
    likes: '999'
};
console.log(usePick(test, [])); // {}
console.log(usePick(test, ['name'])); // {name: '金庸'}
